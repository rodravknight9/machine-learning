# Data Science and Artificial Intelligence

Welcome to the **Data Science and Artificial Intelligence** documentation. This site organizes all course materials and concept notes in one place.

This repository is an **introduction to Data Science and Machine Learning**, built from textbooks, online courses, and hands-on practice—with a little help from AI along the way.

It was used to **dictate a university course** for the subject **Data Science and Artificial Intelligence**, so you'll find lecture-style content, class notes, and practical examples for both self-study and classroom use.

## Browse the docs

Use the **sidebar** (or the tabs above) to jump to:

- **Overview** — machine learning folder README
- **Data preprocessing and visualization** — class doc and techniques summary (snake_case filenames)
- **Linear regression** — comparison of 1D closed-form vs gradient descent

New Markdown files can be included by editing `mkdocs.yml` in the `machine-learning` folder and adding them to the `nav` section (and placing the file under `docs/`).
